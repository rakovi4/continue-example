# Brief Product Description

1. It's a personal Kanban Task Tracker.
2. User manages tasks on a single default board with three columns: To Do, In Progress, Done.
3. Main goal is simple task organization with visual status tracking.
4. The underlying logic is:
   - User creates tasks that land in "To Do" column
   - User moves tasks between columns to track progress
   - User can edit task details or delete completed/obsolete tasks
